To read the RubyFrontier documentation:

(1) Inside the RubyFrontierDocumentation folder, select the "default.txt" file.

(2) Choose RubyFrontier > Publish Site.

(3) Wait a few seconds.

(4) Enjoy!

This folder (RubyFrontierDocumentation) is also offered as an example of a RubyFrontier web site source folder.
